# PCRE Learning Platform

Sistema de aprendizaje de inglés B1 con apuntes en formato PCRE (Pattern, Concept, Rules, Examples) y cuestionarios interactivos.

## Stack Tecnológico

### Frontend
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy + Alembic

## Estructura del Proyecto
```
pcre-learning-platform/
├── apps/
│   ├── frontend/    # Aplicación Next.js
│   └── backend/     # API FastAPI
└── docs/            # Documentación técnica
```

## Desarrollo Local

**Requisitos:**
- Node.js 20+ (LTS)
- Python 3.11+
- PostgreSQL 15+
- pnpm 8+
